Encryption

Encryption is a process that encodes a message or file so that it can be only be read by certain people.Encryption uses an algorithm to scramble, or encrypt, data and then uses a key for the receiving party to unscramble.

Decryption

The conversion of encrypted data into its original form is called Decryption. It is generally a reverse process of encryption. It decodes the encrypted information so that an authorized user can only decrypt the data.
